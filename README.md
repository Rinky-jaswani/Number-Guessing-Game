# Number Guessing Game 🎮

A console-based Number Guessing Game built in Java.

## Features
- Random number generation between 1 and 100
- Smart hints based on how far your guess is
- Scoring system based on attempts used
- Input validation (handles wrong input without crashing)
- Play again option with total score tracking

## Tech Used
- Java (Core)
- Scanner class for user input
- Random class for number generation

## How to Run
javac NumberGuessingGame.java
java NumberGuessingGame

## Concepts Practiced
- Loops and conditionals
- Input handling and validation
- Random number generation
- Score tracking logic
